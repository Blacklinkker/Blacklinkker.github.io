  • Robo para apagar luz sempre que ela é acesa

    Utilizando a placa Arduino uno, um sensor de luminosidade LDR e um Servo-motor, programei a placa para que quando a luz
    estivesse acesa, o Servo-motor a desligaria, com a mesma ideia da The Useless Box.

    Link do Vídeo 

                            
                            #include 
                            Servo servo;
                                
                            int servoPin = 9;
                            int sensorPin = A0;
                                    
                            void setup(){
                                servo.attach(servoPin);
                                servo.write(0);
                            }
                                    
                            void loop(){
                                int luz= analogRead(sensorPin);
                                 if(luz>100){
                                    servo.write(50);
                                    delay(500);
                                    servo.write(0);
                                }
                                delay(100);
                            }
